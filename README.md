# WEZ Maximum Launch Range Estimation using Deep Neural Networks

## Overview

This repository reproduces the methodology described in the paper:

"Maximum Launch Range Estimation using Deep Neural Networks"

The project consists of:

- 3D missile-target engagement simulator
- Augmented Proportional Navigation (APN)
- Binary Search based WEZ estimation
- Latin Hypercube Sampling (LHS)
- Deep Neural Network (TensorFlow)

## Pipeline

Scenario Generation
↓
Missile Simulation
↓
Maximum Launch Range
↓
Dataset Generation
↓
Deep Neural Network
↓
WEZ Prediction
